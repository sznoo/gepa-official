# GPT Actual300 Selected6 Analysis

## Definitions

- `condition`: create_query_hop2 prompt condition.
- `support_recall`: fraction of gold support titles retrieved by hop2 query.
- `missing_recovery`: fraction of missing-after-hop1 titles recovered by hop2 query, recomputed when evaluator field is null.
- `base_answerer_EM`: downstream EM using `base_final_answerer_prompt` on the retrieved context.
- `strong_answerer_EM`: downstream EM using `strong_final_answerer_prompt` on the retrieved context.
- `answerer_delta_EM`: `strong_answerer_EM - base_answerer_EM`; this is answerer-side delta, not prompt-condition gain.
- Paired prompt deltas compare each condition to `condition=base` on the same `(batch, case_id)`.
